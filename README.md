# 🏫 Custom ERP System (College Management Software)

This is a fully modular, lightweight College Management ERP Software built entirely using **HTML5**, **CSS3**, and **Vanilla JavaScript**. It provides a single platform to manage various educational institute departments, including Admin, Accounts, Teachers, Students, and Parents.

The project does not rely on any external frameworks, making it extremely fast and lightweight. All data is saved securely using the browser's `LocalStorage`, allowing the application to work completely offline without requiring any backend database setup.

---

## 🚀 Key Features

*   **Dynamic Course Engine:** Easily add new academic courses from the Admin dashboard. Newly added courses automatically update across the entire system, including admission forms, fees ledgers, and login dropdowns.
*   **Multi-Role Authentication:** Access dedicated portals for Admin, Accountant, Faculty, Student, and Parents through a single gateway.
*   **Automated Fee & Receipt System:** Tracks student fees, features a dynamic UPI QR Code generator, and provides an option to print professional digital receipts.
*   **Bi-Directional Communication:** A built-in messaging/chat bridge to facilitate direct communication between Teachers and Parents.
*   **Report Card Generator:** A comprehensive grades management portal to track and generate internal mid-term and final-term academic results.
*   **Global Notice Board:** An announcement panel allowing the admin to broadcast important circulars and news instantly across all portals.
